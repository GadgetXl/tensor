# Yapay Zeka Modelleri için Tensor Yapısı
Bu proje, Arduino ve ESP32 gibi düşük bellekli cihazlarda yapay zeka modellerini çalıştırmak için geliştirilmiş basit bir tensör yapısı sunar. Tek bir veri yapısı içinde 32-bit float, 16-bit float veya 8-bit integer (quantized) veriler saklanabilir. Quantization ve dequantization ile bellek kullanımı azaltılır.
